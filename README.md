# Netflix Data Cleaning and Preprocessing

This project performs data cleaning and preprocessing on the Netflix dataset using Python and Pandas in Google Colab.

## Steps Performed
- Loaded the dataset
- Checked and handled missing values
- Removed duplicate records
- Standardized column names
- Saved the cleaned dataset

## Tools Used
- Python
- Pandas
- Google Colab
- GitHub

## Output
The cleaned dataset is ready for further analysis and machine learning.
